一、部署麦麦需要的工具

(1) 操作系统要求

本教程环境：
WSL2，默认安装版本为Ubuntu 24.04 LTS，安装步骤请参考官方教程：
如何使用 WSL 在 Windows 上安装 Linux | Microsoft

(2) 开发环境

1.Python解释器：版本3.10及以上（本教程为3.12.6）

2.文本编辑器：本教程使用Visual Studio Code

后面有相当多的操作均可以在VSCode上直接进行，省去部分麻烦

3.数据库：MongoDB

具体如何安装可以查看以下这篇教程：

Windows 平台安装 MongoDB | 菜鸟教程

本教程中仅采用MongoDB本体以及MongoDB的shell工具mongosh，数据可视化用VSCode进行.

4.NapCat QQ Bot

Linux操作系统有一键包安装，可参考官方教程：

Framework | NapCatQQ

注意：请一定要记得将Python解释器以及MongoDB数据库的相关路径加入系统环境变量Path下！！！！
